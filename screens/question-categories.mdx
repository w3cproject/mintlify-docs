## Как получить список категорий вопросов

Получить список категорий вопросов можно с помощью `GET` запроса к API `/question-categories`.

<Card
  title="Get question categories list"
  icon="link"
  href="/api-reference/question-categories/get-question-categories"
>
  Перейти к роуту и изучить основные поля
</Card>

<Note>
  Данный роут **требует** JWT авторизацию для получения данных.
</Note>

### Структура ответа

Метод возвращает массив объектов категорий вопросов:
```json
[
  {
    "id": "int",
    "name": "string"
  },
  {
    "id": "int",
    "name": "string"
  },
]
```
