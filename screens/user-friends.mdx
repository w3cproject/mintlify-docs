## Как получить список друзей

Получить список друзей можно с помощью `GET` запроса к API `/users/friends`.

<Card
  title="Get User friends"
  icon="link"
  href="/api-reference/users/get-user-friends"
>
  Перейти к роуту и изучить основные поля
</Card>

<Note>
  Данный роут **требует** JWT авторизацию для получения данных.
</Note>

### Структура ответа

Метод возвращает массив объектов юзеров

```json
[
  {
    "id": "string",
    "phone": "string",
    "first_name": "string",
    "last_name": "string",
    "avatar_url": "string"
  }
]
```

## Как добавить друзей

Добавить друзей пользователю можно с помощью `POST` запроса к API `/users/friends`.

<Card
  title="Add user friends"
  icon="link"
  href="/api-reference/users/add-user-friends"
>
  Перейти к роуту и изучить основные поля
</Card>

<Note>
  Данный роут **требует** JWT авторизацию для получения данных.
</Note>

### Структура JSON Body

```json
{
  "phones": ["string"]
}
```

где,

`phones` - массив строк, состоящий из телефонов пользователей

### Структура ответа

Метод возвращает массив друзей пользователя с учетом новых

```json
[
  {
    "id": "string",
    "phone": "string",
    "first_name": "string",
    "last_name": "string",
    "avatar_url": "string"
  }
]
```
